# 🤖 AI Chatbot using NLP & Deep Learning

An intelligent AI-based chatbot built using Natural Language Processing (NLP) and Deep Learning (LSTM).  
This chatbot can understand user input, classify intent, and generate appropriate responses.

---

## 🚀 Features

- Intent recognition using NLP
- Text preprocessing (Tokenization, Lemmatization, Stopword removal)
- Deep Learning model using LSTM
- Trained on custom intents dataset
- Real-time chatbot interaction
- Streamlit web interface

---

## 🛠️ Tech Stack

- Python
- TensorFlow / Keras
- NLTK
- NumPy
- Streamlit
- Pickle

---

## 🧠 NLP Pipeline

1. Text Cleaning
2. Tokenization
3. Lemmatization
4. Bag of Words / Sequence Encoding
5. Label Encoding
6. LSTM Model Training
7. Intent Prediction
8. Response Generation

## ▶️ How to Run

### 1️⃣ Install dependencies
pip install -r requirements.txt

### 2️⃣ Train the model (if needed)
python train.py

### 3️⃣ Run the chatbot
streamlit run app.py
